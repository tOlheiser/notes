# Why NoSQL?

Traditional databases are mainly defined for structured data. However, when it comes to 'Big data', defining a well structured way won't work. Alternatively, NoSQL systems can handle both structured and unstructured data in a very efficient manner.

Advantages of NoSQL database systems:
* Consistency
* Scalability/Elasticity
* OOP
* High performance
* Easy integration with server-side languages like PHP and JS
* Economy: Store more data at less cost

## Why MongoDB

The seemingly most popular NoSQL database is MongoDB. It is easy to use, is high performance, high availability, and automatically scales in size.

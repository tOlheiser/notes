5. Set up the routing

## MERN Stack

**MongoDB** - Database Solution. It is a NoSQL DB which stores 'documents' into 'collections' (instead of 'records' in 'tables' in SQL). It enforces no data schema or relations, and it is easily connected to Node/Express. *For security reasons, we don't connect to MongoDB from inside React.*
**Express.js** - Framework for Node.js. Simplifies writing server-side code & logic for Node.js. It is middleware-based, where you funnel requests through functions. It includes Routing, view-rendering, and more.
**React.js** - Browser-side JS library
**Node.js** - Server-side JavaScript Runtime. Gives you the power to listen to requests and send responses, execute server-side logic, and interact with database & files. 

Runtime - Describes the software/instructions that are executed *while* your program is running. Especially those you did not write explicity.

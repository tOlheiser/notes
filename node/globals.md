# Globals
*Nodejs gives you helpful globals. Just like the browser, you should not create your own.*

* **process** - has information about the environment the app is running in.
* **require** - function to fund and use modules in the current module. Require is essentially how you **import** modules.
* **__dirname** - the current directory path.
* **module** - information about the current module, methods for making the module consumable. It is essentially how you **export** modules.

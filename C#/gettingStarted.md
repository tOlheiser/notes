# Getting Started with C#

## Setting up a Console App

1. Create a new project
2. Set you filters to 'C#', 'Windows', and 'Console'
3. Choose 'Console App (.NET Framework)'
4. Leave 'Place solution in project directory' unchecked

The '.sln' file is the top level document for your project. If you want to move your project around, you need to move the whole project folder around.

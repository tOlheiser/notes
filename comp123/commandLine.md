# Command Line Notes

## Commands:

* dir - display a list of files & folders.
* mkdir {directory name} - Create a directory/folder. If you want a directory with more than one word, place the words inside a string - "Movie Files".
* MOVE {directory name} {directory name} - Moves the first directory into the 2nd.
* cd {directory name} - Moves into the specified directory.
* cd .. - Move out of your current directory and into the parent directory.

## Batch Files

A batch file is just a series of commands like those above that you want to execute.

1. Create a text file.
2. Save it as {batchFileName}.bat
3. Double click the batch file to execute.
Done.

# Learning Outcome 1 Exercise 1

## Exercise 1
Write a Java program that prompts the user for a number of tickets. Calculate the total cost of tickets @ 15.99 / ticket. If 10 or more tickets are ordered apply a 15% discount. Store the result in a variable called totalCost and print it to the screen. If there is a discount give, display that discount to the screen along with the price.

## Exercise 2
1.	Write a program that reads an integer and checks whether it is even. 

2.	Write a program that takes as input three integers, sorts them and prints them from smallest to largest.

3.	Write a program that generates two random integers that are less than 100, and asks the user to give the sum of these numbers. The program should then produce an appropriate message.

4.	Write a program that takes as input the length of the three sides of a triangle, checks if the triangle is valid and computes the perimeter if it is. Otherwise display that the input is invalid. (The input is valid if the sum of any of the two sides is greater than the third side. Note that you must check all of the combinations.)

5.	Write a program that generates a random upper case letter. (Generate a random number in the appropriate range and cast it into a char.)

6.	Write a program that simulates the toss of a coin and lets the user guess if it is heads or tails. The program generates an integer (0 or 1) which represents heads for a 1 or tails for a 0. The program prompts the user to enter a guess and reports whether the guess is correct or not.

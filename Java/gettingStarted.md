# Getting Started with Java

*This course uses Netbeans 11.0 + SDK 12.0*

1. Create new project
2. In 'Categories' select 'Java with Ant'

* All Java code is contained in a class.
* Classes should have the same name as their file.

## Main Method

* All Java applications have a main method.
* Methods are sub routines of a larger application.
* Execution always begins at a main method.

```java
public static void main (String[] args) {

}
```

## Data Types

* int - short for integer, which holds negative & positive **whole** numbers.
* double - double precision floating point numbers; *can have decimals*.
* String - holds alpha-numerica data.
* char - Holds a **single** character.
* boolean - Holds either a true or false value. *Can it be truthy or falsey?*

### Examples
```java
int numberOfUnits = 7;
double unitWeight = 4.56;
String unitName = "Widgets";
char shippingCode = 'A';
boolean hasShipped = false;
```

**Operators** - These are identical to the operators I'm used to in JavaScript.
